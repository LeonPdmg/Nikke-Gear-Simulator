# NIKKE Gear Simulator

A interactive, mobile-friendly web application designed to simulate the **Overload Gear Reroll** mechanics from the game *GODDESS OF VICTORY: NIKKE*.

## Features

- **Sequential Slot Rerolling**: Simulates the exact top, middle, and bottom slot rolling probabilities (including lock conditions and duplicate stat exclusion pools).
- **Slot-Specific Odds**: View the exact analytical probability percentages for every stat on a per-slot basis (Top, Middle, Bottom) based on what you have locked.
- **Interactive Phase Reference**: Check weight rates and exact attribute value stats for all Phase levels (1 to 15), featuring custom styled glowing "Black Line" values for Phase 15.
- **Theme Toggle**: Support for a premium tech-dark theme (default) and a clean, high-contrast light theme.
- **Mobile Responsive**: Fully optimized layout for mobile viewports, including vertical stacking comparison views and a compact resource counter.
- **Confirm Before Applying**: Prevents accidental loss of high-tier stats with an option to bypass confirmation for fresh gear.

## Getting Started

Since this is a client-side static web application, no installation or compilation is required.

1. Clone or download the repository.
2. Open `index.html` in any web browser (Chrome, Firefox, Safari, Edge).

Alternatively, you can access the live application directly at https://leonpdmg.github.io/Nikke-Gear-Reroll/

## Files

- `index.html` - Contains the structure and interactive components.
- `index.css` - Custom styling sheet containing layout styles, light/dark theme variables, and tier effects.
- `index.js` - Simulation logic.
